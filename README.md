# FSiAdminTreeBundle

This bundle provides easy way to move up and down node in **Gedmo NestedSet Tree**.
FSiAdminTreeBundle works only with Gedmo Doctrine2 extensions -  NestedSet Tree ``@Tree(type="nested")``
